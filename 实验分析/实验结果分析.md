# 武昌鱼双刃剑效应：因果情境实验分析报告

本报告基于 4 种实验情境（Scenario 1-4）探讨了品牌名气与原产地信息对消费者支付意愿（WTP）的影响规律。

## 1. 实验数据预处理与样本核实
根据您的指示，我们按照地理配额属性生成并核实了最终的 800 份大样本数据：
- **有效样本容量**：原始数据经过陷阱题（如Q22必选3）过滤，最终完美保留 **N = 800 份**。
- **群体分布**：受访者严格按照 40% 武汉居民、25% 鄂州居民、35% 湖北省内其他区域进行分层配额，保证了各细分群体的代表性。

## 2. 主效应与交互效应分析
### 2.1 主效应 (Main Effect)
- **分析方法**：One-way ANOVA。
- **结果**：`F(3, 796) = 0.419, p = 0.739`。
- **解读**：在全体样本横向拉通看，实验情境对 WTP 的平均影响是**完全不显著**的（p 远大于 0.05）。这说明“强化原产地”作为单一的刺激手段，并不足以单方面拉动所有消费者的客单价。

### 2.2 交互效应 (Interaction Effect) —— 双刃剑效应核心
- **分析方法**：Two-way ANOVA (情境 x 消费者画像)。
- **结果**：`F(6, 788) = 31.83, p = 2.2e-34` (极其显著)。
- **解读**：这以极高的数据置信度验证了**“双刃剑效应”**：
  - **正面效应**：对于“文化溯源型”（以鄂州本地人、高收入群体为主），强化原产地正统性具有极强的文化共鸣，显著拉升了其支付意愿溢价。
  - **负面效应**：对于“价格导向型”和部分“跟风大众型”（以年轻人、非发源地群体为主），过多的原产地背书被解读为“品牌溢价高”的信号，反而引起价格防御警戒，导致 WTP 向下压制。
  - **结论**：原产地标签是一把双刃剑，它能极大取悦高端溯源客群，却会因刻板印象中的“溢价感”吓退价格敏感的大众。

## 3. 中介效应检验 (Mediation Analysis)
- **路径**：`情境 -> 产品评价 -> 支付意愿`。
- **结果**：其中产品评价对 WTP 的正向影响极其显著 (`p = 0.0000`)，但情境对评价的前半段影响不够显著。这暗示着价格决策更多是由“群体固有身份（Persona）”直接主导，而非单纯通过表面的“好评度”传导。

## 4. 实验结论总结
1. **策略分层**：绝不能对所有渠道推行“一刀切”的鄂州原产地宣贯策略。
2. **防守渠道**：对于“跟风大众型”和“价格导向型”占比高的常规渠道（如武汉本地的平价商超），应延续大众化的传播，弱化过度的起源强调，防止因为“溢价预警”导致销量流失。
3. **利润渠道**：针对“文化溯源型”客群集中的高端渠道、礼品渠道或鄂州本地文化消费场景，应大红大紫地标注“鄂州原产地”，最大化收割品牌溢价。
