# 基于 K-Means 聚类的武昌鱼消费者画像分析

本分析通过机器学习算法（K-Means），将 N=800 的大样本受访群体在心理量表（产地、文化、品牌、价格）的四维特征上进行收敛聚类，并结合其客观的人口统计学特征（地域、年龄、收入）进行深度交叉映射。

## 1. 聚类方法论与特征降维
### 1.1 变量选取与标准化
选取了四个核心心理维度：
- **产地敏感度**：对原产地真实性的在意程度（Q10-12）。
- **文化涉入度**：对历史文化、三国故事的兴趣程度（Q13-15）。
- **品牌依赖度**：对品牌知名度、行政名片（武汉）的依赖程度（Q16-18）。
- **价格敏感度**：对产品价格及溢价的接受程度（Q19-21）。

所有心理变量均经过 **StandardScaler 标准化处理**，以消除量纲差异，并通过**手肘法** (SSE) 与**轮廓系数** (Silhouette Score) 确定 **k=3** 为最优簇数（见 `optimal_k_evaluation.png` 和 `consumer_segments_radar.png`）。

## 2. 三类人群画像与人口学交叉特征 (Persona & Demographics)
除了纯心理特征之外，我们通过提取 `persona_demographics.png` 的精细数据交叉发现，这三类人群在年龄、收入与地域身份上有着极为清晰的切割边界：

### 2.1 跟风大众型 (Cluster 1, Brand/Follower-oriented)
这类人群很容易受大众声量与知名品牌的影响，买东西倾向随大流，不深究背后的产区知识。

- **核心画像标签**：青年群体、武汉主导、中等收入、强品牌依赖防踩坑。
- **交叉表分布**：
  | 地域属性 | 年龄层级 | 收入画像 |
  | :--- | :--- | :--- |
  | **武汉 (49.3%)**<br>湖北其他 (36.4%)<br>鄂州 (14.3%) | **18-25岁 (59.2%)**<br>**26-35岁 (31.6%)**<br> | 相对均质偏中层：<br>**10k-15k (36.0%)**<br>**6k-10k (28.3%)** |
- **人群解析**：他们是武汉及省内年轻新贵的缩影，拥有充沛购买力但精力有限。买武昌鱼主要依赖“名气”和“大品牌”标签降低试错成本。
- **营销策略**：**包装升维与曝光**。在常规卖场、主流生鲜电商铺满这部分货，主打“高知名度”和“安心购”标签。

### 2.2 价格导向型 (Cluster 2, Price-oriented)
对品牌、文化背景均极度冷感的一群人，决策只看短期性价比。

- **核心画像标签**：极度年轻化/学生、非鄂州本地生源、极致价格敏感。
- **交叉表分布**：
  | 地域属性 | 年龄层级 | 收入画像 |
  | :--- | :--- | :--- |
  | **武汉 (45.1%)**<br>**湖北其他 (44.3%)**<br>鄂州 (10.5%) | **18-25岁 (65.0%)**<br>**<18岁 (23.2%)**<br> | 明显低收入/无产：<br>**学生/无收入 (30.4%)**<br>**3k-6k (29.1%)** |
- **人群解析**：涵盖了庞大的高校学生和初入职场的蓝领白领，几乎囊括了所有 25 岁以下的经济敏感层。对他们灌输高端的“三国底蕴”属于无效动作。
- **营销策略**：**极致性价比卡位**。这是基础销量的基石群体，通过低价爆品抢占市场份额，切忌用高溢价定位吓跑他们。

### 2.3 文化溯源型 (Cluster 3, Culture/Origin-oriented)
极少数但极具价值的金字塔尖客群。他们把武昌鱼视为具备历史谈资和社交价值的文化符号，而不仅仅是食材。

- **核心画像标签**：中年高净值、鄂州本地自豪感、深度文化涉入。
- **交叉表分布**：
  | 地域属性 | 年龄层级 | 收入画像 |
  | :--- | :--- | :--- |
  | **鄂州 (46.7%)**<br>武汉 (27.1%)<br>湖北其他 (26.1%) | **>45岁 (43.3%)**<br>**36-45岁 (37.5%)**<br>26-35 (19.2%) | 绝对的塔尖购买力：<br>**>15k (69.1%)**<br>**10k-15k (30.9%)** |
- **人群解析**：画像结构“老龄化”、“高薪化”且具有极强的“鄂州地域浓度”。这一客群阅历丰富、购买力极强，对鄂州发源地拥有坚定认同，这正是我们在实验中所验证的：对这拨人进行原产地正名教育，可以刺激出极高的支付溢价和忠诚度（详见双刃剑效应）。
- **营销策略**：**原装原产+文化高定**。定制高端伴手礼盒、文化联名款，在高端餐饮、商务答谢渠道深耕这部分高净值客群。
