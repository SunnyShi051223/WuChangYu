# 基于 K-Means 聚类的武昌鱼消费者画像分析

通过对受访者在“产地敏感度”、“文化涉入度”、“品牌依赖度（声望）”以及“价格敏感度”四个维度的加权得分进行 K-Means 聚类分析（k=3），我们将 102 名有效受访者划分为三类典型人群。

## 1. 聚类变量说明
- **产地敏感度**：对“核心原产地”标签的在意程度（Q10-12）。
- **文化涉入度**：对武昌鱼背后的历史、三国文化、诗词的兴趣（Q13-15）。
- **品牌依赖度**：对“名气”、“品牌知名度”以及“行政区域（如武汉）”等外在标签的偏好（Q16-18）。
- **价格敏感度**：对溢价的接受程度及对性价比的追求（Q19-21）。

## 2. 三类典型人群画像 (Persona)

### 2.1 跟风大众型 (Cluster 1 / 跟风大众型) —— 占比约 27%
- **特征描述**：主要由年轻群体（学生）组成。他们对武昌鱼的认知高度依赖于“武汉”名气和“毛主席诗词”等大众化标签。
- **心理动机**：认知惯性极强，不愿深究具体的行政区划，倾向于选择“名气最大”的选项以降低决策成本。
- **策略建议**：**适配“借名”策略**。利用武汉作为省会城市的强符号感进行营销，强化“武昌鱼 = 武汉名片”的关联。

### 2.2 文化溯源型 (Cluster 2 / 文化溯源型) —— 占比约 36%
- **特征描述**：具有较高的文化涉入度和产地敏感度。这部分人群通常为高学历、有稳定收入的社会中坚力量。
- **心理动机**：极其注重品牌的“正统性”与“文化内涵”。他们能够识别并认可“鄂州/梁子湖”作为武昌鱼原产地的稀缺价值，并愿意为此支付一定的文化溢价。
- **策略建议**：**适配“正名”策略**。深挖“梁子湖生态”与“三国孙权命名”的历史背书，针对这部分高端、知识型客户进行精准推广。

### 2.3 价格导向型 (Cluster 3 / 价格导向型) —— 占比约 36%
- **特征描述**：对产地标签和文化故事的敏感度较低（得分均处于中下水平）。
- **心理动机**：核心关注产品的性价比。认为只要口感差别不大，价格是首要考虑因素，对追求“原产地”标签带来的溢价表现冷淡。
- **策略建议**：**稳健供给策略**。在产品设计上应保持基础的高性价比供给，不宜投入过多的文化溢价成本到这部分对成本高度敏感的客群中。

## 3. 统计验证 (见 consumer_segments_radar.png)
通过雷达图可以清晰看到：
- **文化溯源型**的雷达图面积最大，在所有正向价值维度（产地、文化）上均处于领先地位。
- **跟风大众型**在“品牌依赖度”维度上表现突出，但对深层的产地和历史实情涉入不足。
- **价格导向型**各维度得分较为平庸，呈现出明显的低涉入特征。
