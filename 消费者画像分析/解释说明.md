# 基于 K-Means 聚类的武昌鱼消费者画像分析

本分析通过机器学习算法对受访者进行聚类，旨在识别不同的消费者群体及其核心诉求。

## 1. 聚类方法论
### 1.1 变量选取与标准化
选取了四个核心维度作为聚类依据：
- **产地敏感度**：对原产地真实性的在意程度（Q10-12）。
- **文化涉入度**：对历史文化、三国故事的兴趣程度（Q13-15）。
- **品牌依赖度**：对品牌知名度、行政名片（武汉）的依赖程度（Q16-18）。
- **价格敏感度**：对产品价格及溢价的接受程度（Q19-21）。

所有变量均经过 **StandardScaler 标准化处理**，以消除量纲差异对聚类结果的影响。

### 1.2 K 最优值选取 (New)
为了确保聚类的科学性，我们引入了**手肘法 (SSE)** 与 **轮廓系数 (Silhouette Score)** 进行综合评估（见 `optimal_k_evaluation.png`）：
- **手肘法**：在 `k=3` 处斜率发生显著变化，为典型的“手肘点”。
- **轮廓系数**：在 `k=3` 时达到局部峰值，说明簇内紧密度与簇间分离度达到最佳平衡。
- **结论**：最终确定 **k=3** 为本次聚类的最优簇数。

## 2. 三类典型人群画像 (Persona)

根据聚类中心特征值（见雷达图 `consumer_segments_radar.png`），我们将受访者分为以下三类：

### 2.1 跟风大众型 (Cluster 1) —— 占比约 27%
- **核心特征**：高品牌依赖（4.25），对武汉名气认同感强，但产地敏感度较低（3.26）。
- **心理动机**：追求便捷决策，只认“名气”，不深究产地实情。
- **策略暗示**：**适配“借名”策略**。利用武汉省会城市的强符号感，强化产品与城市名片的绑定。

### 2.2 价格导向型 (Cluster 2) —— 占比约 36%
- **核心特征**：各维度得分均较低（产地3.23，文化3.09），核心关注性价比。
- **心理动机**：对品牌文化溢价不敏感，认为口感相近时价格是唯一考量。
- **策略暗示**：**稳健供给策略**。提供高性价比的基础产品，不宜投入过高文化成本。

### 2.3 文化溯源型 (Cluster 3) —— 占比约 36%
- **核心特征**：产地敏感度极高（4.46），文化涉入度极高（4.32），注重正统性。
- **心理动机**：认可“鄂州正统”与“梁子湖生态”价值，追求精神消费与文化认同。
- **策略暗示**：**适配“正名”策略**。深挖三国文化与地理标志背书，针对高端客群进行价值溢价推广。
